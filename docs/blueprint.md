# **App Name**: TeamBook

## Core Features:

- Microsoft Teams Integration: Allow users to sign in and authenticate via their Microsoft Teams accounts.
- Booking Calendar: Display availability slots in a calendar view.
- User Management: Admin will manage user accounts.
- AI Powered Time Suggestion Tool: Suggest the next best available slots based on user calendar and resource availability, using a large language model to reason and make intelligent assumptions when integrating data from different APIs. Incorporates the use of a tool.
- Booking Confirmation: Send confirmation emails via email using Sendgrid/Resend upon successful booking.
- Admin Dashboard: Provide an admin dashboard to view and manage bookings, resources, and users.
- Notifications: Real-time updates to notify changes, confirmations or cancellation.
- Payment Integration: Integrate payment gateway to process payments on booking

## Style Guidelines:

- Primary color: Soft sky blue (#87CEEB), to evoke feelings of trust and dependability.
- Background color: Light grayish-blue (#E6F3FF) to complement the primary color and provide a calm, professional backdrop.
- Accent color: Warm coral (#FF7F50), as it is approximately 30 degrees from blue, which will draw the eye.
- Headline font: 'Poppins' (sans-serif) for a modern, geometric feel. Body font: 'PT Sans' (sans-serif) for readability.
- Use flat, minimalist icons related to bookings and meetings.
- Clean and intuitive layout with clear hierarchy and easy navigation.
- Subtle transitions and animations for a smooth user experience.
